Configurações de RAM
Definição da RAM alocada para o seu jogo, uma quantidade maior aumentará o desempenho do seu jogo.
Observe que você deve evitar adicionar uma quantidade muito grande, caso contrário isso pode causar perda de desempenho em seus outros aplicativos.

-Máximo de RAM: Mostra a quantidade total de RAM alocada para o seu jogo. É recomendado não exceda 50% da quantidade total de RAM presente no seu PC.

-Mínimo de RAM: Indica a quantidade mínima de RAM alocada ao iniciar o jogo. recomendado não colocar mais de 4GB para este valor.

Configurações de resolução do Minecraft
Defina uma resolução personalizada para ter um jogo que caiba na sua tela.
Por padrão o jogo começa em 854 x 480,
você pode modificar este valor para jogar com conforto ótimo. A resolução ideal é 1280 x 720.

Baixar configuração
Esta configuração determina o número máximo de arquivos que podem ser baixados simultaneamente durante o download. atualizar ou baixar recursos.

Observe que alterar esta configuração pode afetar o desempenho do download.
Recomenda-se não exceder os limites padrão recomendados,
a menos que você tenha problemas específicos de conectividade.
